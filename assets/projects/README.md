# Optional project screenshots

Place project preview screenshots in this folder. Recommended filenames:

```text
satellite-monitoring.jpg
food-safety-system.jpg
focusflow.jpg
food-ordering-system.jpg
```

Missing images are safe: the portfolio automatically displays a CSS gradient fallback.
After adding an image, keep its `previewImage` path synchronized in
`assets/data/projects.json` and the fallback array in `script.js`.
